Rental system Car-Rent

Final Year Project

The aim of the project was to build a web application which can be used for renting cars. The application would have different interfaces for an administrator and the user of the system. You can view the application by clicking to this link.

Customers would be capable of viewing and reserving cars by using this system, but visitors must register at the end of their reservation to confirm their booking.
An administrator of the system is responsible for controlling the entire system from adding and deleting cars, price changing, adding, and deleting locations, viewing, and confirming reservations and others.

The above figure is displaying available cars for the chosen Location, Date and Time. As you can see there are four cars available for those specificied details.

The below figure is displaying an administrator control panel. On Admin Panel page, administrator of the system could view all the users, branches, cars and reservations. Moreover, administrator could edit any of the following by just pressing on top their name. As you can see this page is showing all the users and their details.

For the front-end of the application, languages such as HTML, CSS, JavaScript, and Bootstrap been used. For the back-end of the application, languages such as PHP and MySQL have been used. I have used Laravel PHP framework for the project. You can view the application by clicking to this link.
