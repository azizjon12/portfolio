Blog Application Laravel-App

Personal Project

While learning a new programming language called PHP (Hypertext Preprocessor), I was intereseted in learning its frameworks. And, I decided to go with Laravel framework because it was one of the most popular PHP frameworks. Then, I came up with an idea of making my own Blog Application. You can view the application by clicking to this link.

The home page of the Laravel-App shows links to either Register to the application or Login using data such as email address and password. User passwords are stored as hashed on the database of the application.
Users who have not registered to the application can still be able to view the Posts/Blog page beacuse there is no restriction to that page. Anyone who is accessing the application would be able to view all the posts made by the resgistered users.

As you can see from the above figure, registered users can add an image to their posts as well. For instance, if the post been made without an image, application will generate its own image. That image is stored on the server and will be available only when the post been made without an image.

The below figure is displaying a dashboard of the registered user. As you can see, a user named Alex has only 1 post called "Alex's Post". Alex can either Edit or Delete their posts. Moreover, they can create a new post using Create Post link.

For the front-end of the application, languages such as HTML, CSS, JavaScript, and Bootstrap been used. For the back-end of the application, MySQL and Laravel PHP framework have been used for the project. You can view the application by clicking to this link.
